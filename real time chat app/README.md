## Getting Started

### Prerequisites

Node JS

### Installation

1. Set the required environment variables as given in env examples

2. Install NPM packages for frontend and start server

```sh
cd frontend
yarn
yarn dev
```

4. Install NPM packages for backend and start server

```sh
cd backend
yarn
yarn start
```
